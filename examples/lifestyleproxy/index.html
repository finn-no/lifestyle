JSON API proxy

Super dumb HTTP API. Provides 4 read-only resources that deliver JSON reprsentations of some of the resources the full REST API exposes:
/searches
A list of search IDs
/search
Perform a search. Takes the query argument "searchid", which must be a valid search ID. Any other query arguments are passed on to the API client as search arguments.
/description
Get an OpenSearch search description. Takes the query argument "searchid", which must be a valid search ID.
/ad
An ad. Takes a single adid query argument.
